Principiul fizic

O prismă optică produce două fenomene importante:

1. Refracția

Lumina își schimbă direcția la trecerea dintr-un mediu transparent în altul.

2. Dispersia

Lumina albă se separă în culori, deoarece fiecare culoare se refractă puțin diferit.

$n_1\sin i=n_2\sin r$

Aceasta este legea refracției. În prismă, raza se refractă la prima față și apoi încă o dată la a doua față.

Ce este prisma optică?

O prismă optică este un corp transparent, mărginit de fețe plane, care poate devia lumina.

În lecțiile de bază folosim de obicei o prismă triunghiulară. Secțiunea ei are forma unui triunghi.

În desen, raza emergentă nu mai are aceeași direcție cu raza incidentă. Ea este deviată.

De ce se deviază lumina?

Prisma are două suprafețe de separare aer–sticlă și sticlă–aer. La fiecare suprafață are loc refracție.

  1. La intrarea în prismă, lumina trece din aer în sticlă și se schimbă viteza de propagare.
  2. În interiorul prismei, raza merge pe o nouă direcție.
  3. La ieșire, lumina trece din sticlă în aer și se refractă din nou.

Rezultatul celor două refracții este o abatere totală a razei față de direcția inițială.

Pentru o prismă triunghiulară aflată în aer, raza este deviată, în general, spre baza prismei.

Lumina albă nu este o singură culoare

Lumina albă, cum este lumina Soarelui, este un amestec de mai multe radiații colorate.

roșu orange galben verde albastru indigo violet

Când lumina albă trece prin prismă, culorile se separă. Tabloul de culori obținut se numește spectru.

Dispersia luminii

Dispersia luminii este fenomenul prin care lumina albă se descompune în culorile componente la trecerea printr-un mediu dispersiv, cum este sticla prismei.

lumină albă $\longrightarrow$ roșu + orange + galben + verde + albastru + indigo + violet

Fizic, explicația este aceasta: indicele de refracție al sticlei nu este exact același pentru toate culorile.

De aceea, fiecare culoare iese din prismă pe o direcție ușor diferită.

Legătura cu refracția

În lecția despre refracție ai învățat că direcția razei se schimbă din cauza schimbării vitezei luminii în alt mediu.

$n=\dfrac{c}{v}$

La dispersie apare un detaliu nou: în sticlă, viteza luminii depinde ușor de culoare.

Ideea-cheie: dacă indicele de refracție este puțin diferit pentru fiecare culoare, și unghiul de refracție va fi puțin diferit.

Astfel, un singur fascicul alb poate deveni un evantai de raze colorate.

Experiment simplu

Poți observa efecte asemănătoare dispersiei în situații cotidiene.

  1. Privește lumina Soarelui care trece printr-un pahar cu apă.
  2. Așază o coală albă în dreptul luminii ieșite.
  3. Schimbă încet poziția paharului.
  4. Caută apariția unor pete sau margini colorate.

Atenție: nu privi direct spre Soare. Observă doar lumina proiectată pe hârtie sau pe perete.

Un curcubeu apare tot prin refracție, reflexie internă și dispersie în picăturile de apă din atmosferă.

Aplicație: spectroscopul

Prisma poate fi folosită pentru a analiza lumina emisă de diferite surse.

Un spectroscop este un aparat care permite observarea vizuală a spectrelor.

Cu ajutorul spectrului putem afla informații despre sursa de lumină. De exemplu, astronomii folosesc spectrele pentru a studia stelele.

Exemplu rezolvat pas cu pas

Problemă: O rază de lumină albă cade pe o prismă de sticlă. Pe ecran apar mai multe culori. Care culoare este deviată mai mult: roșu sau violet?

  1. Principiul: prisma produce refracție și dispersie.
  2. Dispersia: culorile se separă deoarece se refractă diferit.
  3. Comparația: în sticlă, violetul se abate mai mult decât roșul.

Răspuns: raza violetă este deviată mai mult decât raza roșie.

Verifică-te

De ce nu spunem că prisma „creează” culorile?

Răspuns: deoarece culorile erau deja componente ale luminii albe; prisma doar le separă.

Recapitulare

  • Prisma optică este un corp transparent care deviază lumina prin refracție.
  • O prismă triunghiulară aflată în aer deviază raza, în general, spre baza prismei.
  • Lumina albă este un amestec de radiații colorate.
  • Dispersia este descompunerea luminii albe în culorile componente.
  • Spectrul este tabloul culorilor obținut după dispersie.
  • Roșul se abate mai puțin, iar violetul se abate mai mult.
  • Dispersia apare deoarece indicele de refracție depinde de culoarea luminii.
