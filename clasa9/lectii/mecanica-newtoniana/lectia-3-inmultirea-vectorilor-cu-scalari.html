Lecția 3

Înmulțirea vectorilor cu scalari

Ce învățăm?

În această lecție aflăm ce se întâmplă cu un vector atunci când este înmulțit cu un număr real: când se lungește, când se micșorează, când își schimbă sensul și când devine vector nul.

Întrebare de pornire

Dacă o forță $\vec{F}$ este înlocuită cu $2\vec{F}$, ce se schimbă: valoarea, direcția sau sensul ei?

1. Ideea fizică

Un vector descrie o mărime care are modul, direcție și sens. Un scalar este un număr real, cum ar fi $2$, $\frac{1}{3}$, $-1$ sau $0$.

$\vec{b}=k\vec{a}$

Când înmulțim vectorul $\vec{a}$ cu scalarul $k$, obținem un vector nou, notat $k\vec{a}$.

2. Regula de bază

Scalarul $k$ modifică modulul vectorului. Semnul lui $k$ decide sensul.

Valoarea lui $k$ Ce se întâmplă cu vectorul $k\vec{a}$?
$k>0$ are același sens cu $\vec{a}$
$k<0$ are sens opus față de $\vec{a}$
$k=0$ devine vectorul nul $\vec{0}$

$|k\vec{a}|=|k|\cdot |\vec{a}|$

3. Înmulțirea cu un număr pozitiv

Dacă $k$ este pozitiv, vectorul își păstrează direcția și sensul. Se schimbă doar lungimea lui.

Exemplu: dacă $|\vec{a}|=5\,\text{N}$, atunci $|2\vec{a}|=10\,\text{N}$.

4. Înmulțirea cu o fracție pozitivă

Dacă $0<k<1$, vectorul devine mai scurt, dar direcția și sensul rămân aceleași.

$\left|\frac{1}{2}\vec{a}\right|=\frac{1}{2}|\vec{a}|$

5. Înmulțirea cu un număr negativ

Dacă $k$ este negativ, vectorul rezultat are aceeași direcție, dar sens opus. Modulul se înmulțește cu $|k|$.

Vectorul $-\vec{a}$ se numește opusul vectorului $\vec{a}$.

6. Ce înseamnă $-2\vec{a}$?

Semnul minus schimbă sensul, iar numărul $2$ dublează modulul.

$|-2\vec{a}|=2|\vec{a}|$

Direcția rămâne aceeași dreaptă-suport, sensul se inversează, iar lungimea se dublează.

7. Înmulțirea cu zero

Dacă înmulțim orice vector cu scalarul $0$, obținem vectorul nul.

$0\cdot \vec{a}=\vec{0}$

Modul

$|\vec{0}|=0$

Observație

Vectorul nul nu are un sens fizic bine determinat.

În fizică, $\vec{0}$ poate însemna absența unei mărimi vectoriale: forță rezultantă nulă, deplasare nulă sau viteză nulă.

8. Interpretare pe axă

Pe o axă orientată, semnul arată sensul, iar valoarea absolută arată de câte ori se mărește sau se micșorează vectorul.

9. Exemplu fizic: forța

O forță este vector. Dacă asupra unui corp acționează o forță $\vec{F}$, atunci $3\vec{F}$ înseamnă o forță de trei ori mai mare, pe aceeași direcție și în același sens.

Problemă ghidată

Se dă $|\vec{F}|=4\,\text{N}$. Determină modulul vectorului $3\vec{F}$.

Rezolvare pas cu pas

Aplicăm formula $|k\vec{F}|=|k|\cdot|\vec{F}|$.

$|3\vec{F}|=3\cdot 4\,\text{N}=12\,\text{N}$.

10. Exemplu fizic: viteza

Viteza este vector. Dacă $\vec{v}$ este orientată spre est, atunci $-\vec{v}$ este orientată spre vest, având același modul.

Mișcare spre vest, cu aceeași valoare a vitezei.

11. Proprietăți utile

Înmulțirea vectorilor cu scalari respectă reguli asemănătoare cu cele din algebră, dar trebuie să păstrăm interpretarea geometrică.

$(mn)\vec{a}=m(n\vec{a})$

$(m+n)\vec{a}=m\vec{a}+n\vec{a}$

$m(\vec{a}+\vec{b})=m\vec{a}+m\vec{b}$

Aceste proprietăți sunt foarte utile când vom descompune vectori pe axe sau când vom lucra cu forțe.

12. Greșeli frecvente

Greșeală

„Dacă $k$ este negativ, direcția vectorului se schimbă.”

Corect

Direcția rămâne aceeași. Se schimbă doar sensul.

Greșeală

„$-2\vec{a}$ are modul negativ.”

Corect

Modulul este mereu pozitiv sau zero: $|-2\vec{a}|=2|\vec{a}|$.

13. Exersează

Exercițiul 1

Pentru un vector $\vec{a}$ cu $|\vec{a}|=6\,\text{m}$, determină modulele: $2\vec{a}$, $\frac{1}{3}\vec{a}$ și $-\vec{a}$.

Răspuns ghidat

$|2\vec{a}|=12\,\text{m}$, $\left|\frac{1}{3}\vec{a}\right|=2\,\text{m}$, $|-\vec{a}|=6\,\text{m}$.

La $-\vec{a}$ se schimbă sensul, nu modulul.

Exercițiul 2

Desenează un vector $\vec{u}$ spre nord și apoi vectorul $-3\vec{u}$. Ce trebuie să modifici?

14. Recapitulare

  • $k\vec{a}$ este un vector obținut din $\vec{a}$ prin înmulțirea cu scalarul $k$.
  • Modulul se calculează cu $|k\vec{a}|=|k|\cdot|\vec{a}|$.
  • Dacă $k>0$, sensul rămâne același.
  • Dacă $k<0$, sensul se inversează.
  • Dacă $k=0$, rezultatul este vectorul nul $\vec{0}$.
